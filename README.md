# Python Learning Journey

## 關於我

我是一名資訊管理研究生，預計 2026 年 6 月底會從國立彰化師範大學資訊管理碩士畢業。目前正在從零開始加強 Python 程式能力。希望可以持續精進到成為一名合格的軟體工程師。

## 為什麼開始學 Python

我希望未來能具備軟體工程師與資料工程師所需要的基礎程式能力，因此建立這個 Repository，記錄自己的 Python 學習過程。並且讓自己養成每天學習程式的習慣直到我具備應徵科技業軟體工程師職缺的能力。

## 學習目標

- 熟悉 Python 基礎語法
- 學會使用 VS Code 與終端機
- 學會使用 Git 與 GitHub 管理學習紀錄
- 逐步建立可以展示的程式作品

## Phase 0 完成內容

- 安裝並確認 Python 可正常執行
- 安裝並設定 VS Code
- 建立 `python-learning` 專案資料夾
- 建立並執行第一個 Python 程式 `hello.py`
- 學會基本終端機指令
- 學會使用 Git 建立第一次 commit
- 成功將專案 push 到 GitHub

## 下一步

接下來我會進入 Phase 1，開始學習 Python 基礎語法，例如變數、資料型態、輸入輸出與條件判斷。我希望透過紮實的學習來讓我成為一名合格的軟體工程師。
